# Задача 25.2: Перевірка синтаксису виразу

**Що робить програма:**
Клієнт відправляє серверу арифметичний вираз, а сервер перевіряє його на синтаксичну правильність.

**Як запустити:**
1. Запустити `server.py`.
2. Запустити `client.py` в іншому терміналі.
3. Ввести вираз у консолі клієнта.

**Приклад вводу/виводу:**
- Ввід: `2 - 57*(33 + 25/4)`
- Вивід сервера: `Синтаксис правильний`
