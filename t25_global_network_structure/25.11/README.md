# Задача 25.11: Чат між двома клієнтами

**Що робить програма:**
Сервер дозволяє двом клієнтам обмінюватися текстовими повідомленнями по черзі.

**Як запустити:**
1. Запустіть `server.py`.
2. Запустіть два окремі вікна термінала і в кожному запустіть `client.py`.
3. Перший клієнт пише повідомлення, другий його отримує і пише у відповідь.

**Приклад вводу/виводу:**
- Клієнт 1: `Привіт!`
- Клієнт 2 бачить: `Співрозмовник: Привіт!`
