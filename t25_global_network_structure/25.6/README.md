# Задача 25.6: Обмін файлами через сокети

**Що робить програма:**
Програма реалізує клієнт-серверну взаємодію для передачі файлів. Клієнт підключається до сервера, відправляє назву файлу, а потім його вміст у бінарному форматі. Сервер приймає дані та зберігає їх у папку `received_files`.

**Як запустити:**
1. Переконайтеся, що в папці із задачею є текстовий файл для тесту (наприклад, `test.txt`).
2. Запустіть сервер:
   ```bash
   python server.py
