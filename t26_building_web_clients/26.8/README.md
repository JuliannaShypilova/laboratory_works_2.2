# Програма отримання розкладу занять (КНУ ім. Шевченка)

## Опис проекту
Даний скрипт автоматично завантажує актуальний файл розкладу з офіційного сайту механіко-математичного факультету КНУ та виводить пари для конкретної спеціальності, курсу та дня тижня.

## Основні функції
- Автоматичний пошук посилання на .xlsx файл на сайті.
- Фільтрація розкладу за назвою освітньої програми (наприклад, Математика, Статистика тощо).
- Врахування поділу на підгрупи.
- Очищення виводу від "чужих" пар, які записані в спільних клітинках Excel.

## Технології
- **Python 3.x**
- **Pandas** — для обробки таблиць Excel.
- **BeautifulSoup4** — для парсингу веб-сторінки.
- **Requests** — для завантаження файлів.
