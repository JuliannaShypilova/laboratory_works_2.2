# Задача 31.3: Синхронізація потоків (Генератор-Обробник)

Дана програма реалізує модель взаємодії двох потоків через спільну чергу.

## Логіка роботи:
* **Потік-генератор:** створює повідомлення з випадковим інтервалом.
* **Потік-обробник:** виймає повідомлення з черги та виводить на екран.
* **Синхронізація:** наступне повідомлення генерується тільки після того, як обробник підтвердив завершення роботи з попереднім (використано `threading.Event`).

## Стек:
* Python 3.x
* Модулі: `threading`, `queue`, `time`, `random`
