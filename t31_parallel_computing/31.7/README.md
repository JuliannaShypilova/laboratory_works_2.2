# Задача 31.7: Моделювання системи черг у супермаркеті

Програма імітує роботу касової зони з динамічним управлінням ресурсами (чергами).

## Опис моделі:
* **Касири (Потоки):** Кожна каса працює як окремий потік (`threading.Thread`).
* **Покупці:** Приходять через випадкові інтервали та обирають найкоротшу чергу.
* **Резервні каси:** Якщо довжина черги до будь-якої каси перевищує ліміт **L_max**, система автоматично відкриває нову (резервну) касу.
* **Перерви:** Реалізовано випадковий вихід касирів на перерву, що імітує реальні затримки в обслуговуванні.

## Технічні деталі:
* Використано безпечні для потоків черги `queue.Queue`.
* Параметри моделювання (N, M, K, T1-T4) легко налаштовуються в коді для аналізу пропускної здатності системи.
